# Contributing

Thanks for your interest in the Tabnine plugin for opencode.

This repository is for **releases** and **issue tracking** only. It does not
contain the product source code and does not accept code contributions, so
**pull requests are not accepted** and will be closed automatically.

Here is how to help:

- **Found a bug?** [Open a bug report](https://github.com/codota/tabnine-opencode-public/issues/new?template=bug_report.yml).
- **Have an idea?** [Open a feature request](https://github.com/codota/tabnine-opencode-public/issues/new?template=feature_request.yml).
- **Security issue?** Please report it privately, see [SECURITY.md](SECURITY.md).

For account, licensing, or deployment questions, contact your Tabnine
administrator or Tabnine support at support@tabnine.com.
