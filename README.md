# Tabnine plugin for opencode

Public home for **releases** and **issue tracking** for the Tabnine plugin for opencode (`tabnine-opencode`), Tabnine's distribution of the opencode CLI coding agent.

> **Beta** · current version: 0.4.0

This repository does not contain the product source code. It exists so you can download releases and report problems.

## Download and install

Get the installer for your platform from the [Releases page](https://github.com/codota/tabnine-opencode-public/releases).

Installation, sign-in, and usage instructions are in the documentation:

- **Documentation:** https://docs.tabnine.com

Supported platforms: macOS (x64, arm64), Linux (x64, arm64), and Windows (x64, arm64).

## Report a bug or request a feature

Please use the issue templates:

- [Report a bug](https://github.com/codota/tabnine-opencode-public/issues/new?template=bug_report.yml)
- [Request a feature](https://github.com/codota/tabnine-opencode-public/issues/new?template=feature_request.yml)

Before opening an issue, search [existing issues](https://github.com/codota/tabnine-opencode-public/issues) to avoid duplicates. Bug reports are much easier to act on when they include your version (`tabnine-opencode --version`), your operating system, and clear steps to reproduce.

## Security

Do not report security vulnerabilities through public issues. See [SECURITY.md](SECURITY.md) for how to report them privately.

## Support

For account, licensing, or deployment questions, contact your Tabnine administrator or Tabnine support at support@tabnine.com.
